Umbrella is a lightweight macro utility designed to streamline repetitive in-game inputs and improve workflow efficiency for players of Dota 2. The goal of the project is to provide a clean, minimal, and performance-friendly assistant that focuses on input automation without modifying game files or interacting with protected memory.

Umbrella works as an external macro tool. It operates at the input level only, meaning it simulates standard keyboard and mouse actions in a controlled and user-configurable way. The application does not inject into the game process and does not alter gameplay data. It is built with stability and safety in mind, ensuring compatibility with standard system environments.
